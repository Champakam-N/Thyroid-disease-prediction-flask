About the Image-Based Diagnosis of Thyroid Disorders

Dataset Overview

The model is trained on a curated dataset of thyroid ultrasound images collected from verified medical sources. The dataset includes three primary categories — Normal, Hypothyroidism, and Hyperthyroidism. Each image is resized to 128×128 pixels and preprocessed using normalization and augmentation techniques to enhance model robustness and prevent overfitting.

CNN Model Architecture

The system employs a Convolutional Neural Network (CNN) with multiple convolutional and pooling layers followed by dense layers for final classification. The model outputs three classes using a Softmax layer and achieves high accuracy with Adam optimization and early stopping callbacks during training.

Prediction and Confidence

During prediction, the user selects an image, which is preprocessed and fed into the trained model. The system identifies the thyroid condition and displays the predicted class along with a confidence percentage. This helps clinicians and patients quickly assess diagnostic probabilities.

Interactive GUI

The Tkinter-based desktop GUI enables users to easily upload thyroid scan images. It displays the selected image, prediction result, and confidence score in a clean and simple layout. The system also allows smooth operation with "Select Image" and "Cancel" options for user convenience.
